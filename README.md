# GlowStack Pricing Cards

## Description
GlowStack is a modern web application designed to showcase various services and pricing plans. The application features an intuitive user interface with smooth navigation, responsive design, and interactive elements. It aims to empower innovation with smart tech stacks, providing users with a clear understanding of the services offered and their respective pricing.

## Features
- **Responsive Design**: Adapts seamlessly to different screen sizes.
- **Interactive Pricing Cards**: Users can toggle between monthly and yearly pricing plans.
- **Smooth Navigation**: Easy access to different sections of the application.
- **FAQ Section**: Expandable questions and answers for user convenience.
- **Dark Mode**: A toggle switch to switch between light and dark themes.
- **Contact Form**: Users can reach out for inquiries or support.

## Technologies Used
- HTML
- CSS
- JavaScript
- Tailwind CSS
- AOS (Animate On Scroll)

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/glowstack-pricing-cards.git

2. Navigate to the project directory:
   ```bash
   cd glowstack-pricing-cards
Open the index.html file in your web browser.

## Usage
1. Navigate through the application using the header links.
2. Explore the services offered in the "Our Services" section.
3. View and toggle between pricing plans in the "Pricing" section.
4. Expand the FAQ items to read answers to common questions.
5. Use the contact form to send inquiries.
